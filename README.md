# Helpora v1.0 GitHub Pages Website

Updates included:
- Removed Healthcare and Animal Welfare causes.
- Added About Us menu and page.
- Added founder story / defining moment section.
- Added Trust Center page.
- Added modern colorful homepage design.
- Added placeholders for founder photo.

To add founder photo:
1. Save the photo as `founder.jpg`.
2. Upload it to `assets/images/`.
3. In `about.html`, replace the founder placeholder block with:
   `<img src="assets/images/founder.jpg" alt="Founder of Helpora">`

Deploy using GitHub Pages: Settings → Pages → Deploy from branch → main → root.
